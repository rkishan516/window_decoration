# Changelog

## [Unreleased]

### Changed
- Migrated to Dart workspace architecture

## [0.1.0] - 2025-01-16

### Added
- Initial release with full macOS support
- Window positioning (center, getBounds, setBounds)
- Window appearance (background color, opacity)
- Window behavior (always on top, skip taskbar, fullscreen)
- Title bar styles (normal, hidden, transparent, unified)
- NSVisualEffectMaterial support for vibrancy
- Window shadow control
- Movable by window background
- Collection behavior configuration
